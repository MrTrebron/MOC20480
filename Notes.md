## Javascript
### `var` vs. `let`
Eine Variable mit `var` ist in der ganzen Funktion gültig. Auch bei eingeschobenen Code Blöcken
Eine Variable mit `let` ist nur in diesem Kontext gültig. Bei engeschobenen Code Block
